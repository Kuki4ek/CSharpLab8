# Обработка XML документов.
Считать автоматически цвета с изображения палитры (https://pxc-spb.ru/img_catalog/komlekt-krovla/dobor/RAL_2.gif) и представить эту палитру в формате XML-документа. С использованием библиотеки для оптического распознавания текста (OCR) (например, Puma.NET или Tesseract) код цвета нужно так же определить и записать в XML-файл. Потом этот документ необходимо считать и вывести палитру на его основе. Палитра выводится в
файл, при этом пользователь может выбрать количество столбцов, а так же ширину и высоту одного прямоугольника с цветом и расстояния между ними. Помимо предложенной, можно использовать другие палитры с сайта http://www.pxc-spb.ru/img_catalog/komlekt-krovla/dobor/ral.html.
